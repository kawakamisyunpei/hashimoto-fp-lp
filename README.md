# 橋本一利 プロフィールLP

Appleライクな1ページ構成のプロフィールLP。バニラ HTML/CSS/JS（ビルド不要）。
`index.html` をブラウザで開くだけで表示できます。

## ファイル構成
```
8_橋本FP_プロフィールLP/
├─ index.html          … LP本体（全セクション）
├─ css/
│  ├─ style.css        … 本デザイン（使用中）
│  └─ wireframe.css    … フェーズ1のワイヤーフレーム（参考・未使用）
├─ js/
│  └─ main.js          … 動き（ヘッダー・スクロール出現）
├─ images/             … 写真・イラスト
└─ README.md
```

## 制作フェーズ
1. ワイヤーフレーム（構成確定）
2. CSSデザイン（Appleライク・白基調・実写メイン＋イラストアクセント）
3. JavaScript（スクロールで各セクションがふわっと出現／ヘッダーのスクロール状態／ヒーローの読み込みアニメ／`prefers-reduced-motion`対応）

## 画像の使い方
| ファイル | 使用場所 |
|---|---|
| 13039095-…jpg（屋外・立ち姿） | ヒーロー背景 |
| 03F2A3E0-…jpg（着席ポートレート） | プロフィール |
| 2CBDD8E5-…jpg（相談シーン） | 相談風景の帯 |
| FC274D71-…jpg（イラスト切り抜き） | LINE CTAの円形アバター |
| 414501A4-…png（屋外イラスト） | **未使用**（ヒーロー横やセクション背景に追加可） |

写真を差し替える場合は `images/` に置き、`index.html` の該当 `src` を書き換えてください。

## 差し替えが必要なプレースホルダー
現在ダミー（`href="#"` など）になっている箇所：
- **LINE公式アカウントURL** … `index.html` 内 `href="#line"` のCTAリンク、および LINE CTAボタン `href="#"`
- **Instagram URL** … フッター `footer__sns`
- **PDFプレゼント4種の実ファイル** … LINE登録特典（家計簿テンプレート／ライフプランシート／教育資金ガイド／資産運用チェックリスト）
- 電話番号など連絡先を載せる場合は別途追記

## カラー（CSS変数：`css/style.css` 先頭 `:root`）
- つかう＝ブルー `--use` / ふやす＝グリーン `--grow` / まもる＝ゴールド `--protect`
- 背景の温かいオフホワイト `--bg-warm`、アクセントゴールド `--gold`、LINE緑 `--line-green`
